Розробити простий консольний додаток, яке запитує назву міста, та видає детальну відповідь по погоді.

Отримати поточну погоду за вказаним містом через відкрите API (https://wttr.in/). 

	2.1 Отримувати інформацію з API (https://wttr.in/Poltava?) 
	2.2 Відобразити температуру, стан погоди та іконку у GUI. 
	2.3 Оновлювати дані за кнопкою та з таймером. 
	2.4 Показати в інтерфейсі:

		- Температура. 
		- Місто. 
		- Значок (на свій смак).

Валідація міста (перевірити, чи wttr.in розпізнало його)

Автоматичне визначення міста (Якщо не вказано місто — викликаємо https://wttr.in/?format=j1, і воно автоматично визначить за IP.)

Парсинг опису стану для локалізації (по типу weatherDesc[0].value = "Partly cloudy" → "Мінлива хмарність")

Додати кешування (Щоб не дратувати API, додай кеш із часом оновлення.)

Обробка помилок мережі та некоректних відповідей

- Перевірка статусу HTTP

- Обробка null, відсутніх полів

- Обробка HttpRequestException, JsonException
